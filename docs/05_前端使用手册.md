# 前端使用手册

## 1. 打开页面

启动服务：

```bash
python web/app.py
```

浏览器访问：

```text
http://127.0.0.1:8000
```

打开后先看顶部运行模式区域。

## 2. 先理解当前页面

当前页面分成：

- 顶部状态区
- 左侧三模块工作区
- 右侧智能助手

左侧模块不是跳转页面，而是页内切换：

- 视频分析
- 内容创作
- 知识库管理

## 3. 视频分析怎么用

### 操作步骤

1. 切到 `视频分析`
2. 粘贴 B 站视频链接
3. 等待系统自动解析视频信息
4. 确认预览区已出现标题、UP 主、播放等信息
5. 点击 `一键解析并分析视频`

### 结果区会看到什么

- 自动解析后的视频信息
- 热门爆款 / 播放偏低判断
- 原因分析
- 优化建议
- 后续选题建议
- 参考视频
- 对低表现视频生成的一版新文案

### 使用注意

- 当前页面已经是“先自动解析，再点击分析”
- 不是旧版“点按钮后才开始解析”

## 4. 内容创作怎么用

### 操作步骤

1. 切到 `内容创作`
2. 输入领域、方向、想法中的任意一项或多项
3. 选择内容分区
4. 选择文案风格
5. 点击 `一键生成选题与文案`

### 结果区会看到什么

- 整理后的方向
- 当前核心问题
- 推荐主选题
- 选题建议列表
- 标题
- 分段脚本
- 简介
- 标签
- 置顶评论

### 复制能力

结果区支持直接复制：

- 标题集合
- 单个标题
- 完整脚本
- 单个脚本片段
- 简介
- 置顶评论

## 5. 知识库管理怎么用

知识库模块有 4 个子功能。

### 5.1 上传知识文件

操作步骤：

1. 切到 `知识库管理`
2. 保持子 Tab 在 `上传知识文件`
3. 选择本地文件
4. 点击上传

当前支持格式：

- `.txt`
- `.md`
- `.docx`
- `.pdf`

### 5.2 同步热门样本

操作步骤：

1. 切到子 Tab `同步热门样本`
2. 设置每个榜单同步条数
3. 点击 `自动更新热门知识库`

当前同步内容包括：

- 全站热门榜
- 每周必看
- 入站必刷
- 真实一级分区热门榜

如果你刚修改了同步逻辑：

- 先重启后端
- 再点同步

### 5.3 查看知识库

操作步骤：

1. 切到子 Tab `查看知识库`
2. 设置查看条数
3. 点击 `查看知识库内容`

### 5.4 关键词检索

操作步骤：

1. 切到子 Tab `关键词检索`
2. 从下拉框里选择一级分类
3. 点击检索

当前检索不是细分赛道下拉，而是一级分类下拉。

例如：

- 你点 `舞蹈`
- 后端会先召回
- 再把不属于舞蹈类的命中过滤掉
- 最终只保留更符合该分类的结果

## 6. 智能助手怎么用

### 前提

必须先开启 `LLM Agent 模式`。

### 操作步骤

1. 在顶部运行模式区域开启开关
2. 如果没有已保存配置，先填写：
   - LLM URL
   - LLM Key
   - 模型供应商
   - 模型名称
3. 保存配置
4. 在右侧输入问题并发送

### 适合直接问的问题

- 帮我分析这个视频为什么没起量
- 帮我出 3 个后续选题
- 帮我写一版更像生活区 vlog 的标题
- 帮我看看最近热点里什么题材更适合我

## 7. 常见问题

### 7.1 为什么右侧助手点不了

原因通常是：

- 当前还是无 Key 逻辑模式
- 或者还没有可用 LLM 配置

### 7.2 为什么知识库检索结果看起来分类不对

先确认两件事：

1. 你点的是不是一级分类
2. 热门样本是不是已经按新逻辑重新同步过

### 7.3 为什么热门样本同步后还是旧分区

这通常说明：

- 后端还没重启
- 或者同步前库里还是旧样本，需要重新跑一次同步

## 8. 当前版本的重要变化

- 页面已经是三模块，不是双模块
- 知识库已成为正式模块
- 检索下拉只保留一级分类
- 热门样本标题展示已去掉原始 `BV...` 文档 ID
- 搜索结果会做去重，不再把多个分片拆成多张卡片
