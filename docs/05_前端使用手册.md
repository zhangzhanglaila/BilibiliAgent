# 前端使用手册

## 1. 怎么打开页面

启动服务：

```bash
python web/app.py
```

浏览器访问：

```text
http://127.0.0.1:8000
```

打开页面后，先看顶部的运行模式说明。

你会看到两种之一：

- `无 Key 规则模式`
- `LLM Agent 模式`

如果你想切模式，见：

- [07_运行模式切换说明.md](D:/agent/docs/07_运行模式切换说明.md)

## 2. 页面怎么理解

当前页面分成 3 部分：

- 顶部状态区
- 左侧业务区
- 右侧智能助手

左侧不是两个页面跳转，而是两个标签切换：

- `视频分析`
- `内容创作`

## 3. 内容创作模块怎么用

适合：

- 还不知道做什么内容
- 已经有方向，但不知道怎么整理成更容易起量的选题
- 想直接生成标题、脚本、简介和标签

### 3.1 操作步骤

1. 点击左侧标签切到 `内容创作`
2. 输入以下任意 1 项或多项：
   - 你的领域
   - 你想做的方向
   - 你已有的想法
3. 选择内容分区
4. 选择文案风格
5. 点击“一键生成选题与文案”

### 3.2 结果区会看到什么

你会看到：

- 整理后的方向
- 当前核心问题
- 推荐主选题
- 3 个选题建议
- 标题集合
- 分段脚本
- 视频简介
- 标签
- 置顶评论

### 3.3 可直接做什么

结果页里很多内容都可以直接复制：

- 标题集合
- 单个标题
- 完整脚本
- 单个脚本片段
- 简介
- 置顶评论

## 4. 视频分析模块怎么用

适合：

- 视频已经发布
- 想知道当前更像爆款还是播放偏低
- 想知道下一步怎么优化

### 4.1 操作步骤

1. 点击左侧标签切到 `视频分析`
2. 把 B 站视频链接粘贴到输入框
3. 等待系统先自动解析视频
4. 确认预览区已经显示标题、UP 主、播放等信息
5. 点击“一键解析并分析视频”

### 4.2 一个非常重要的变化

当前页面已经改成：

- 先自动解析
- 再点击分析

不是旧版那种“点按钮才开始解析”。

### 4.3 支持什么链接

当前支持：

- 标准 B 站视频链接
- `b23.tv` 短链

### 4.4 分析后会看到什么

分析结果区会展示：

- 当前视频核心信息
- 热门爆款 / 播放偏低判断
- 原因分析
- 后续选题建议
- 标题、封面、内容优化建议
- 低表现时的新文案
- 可直接点击的参考视频

## 5. 智能对话助手怎么用

适合：

- 不想自己判断该用哪个模块
- 想直接自然语言提问
- 想让系统自行调用工具完成任务

### 5.1 使用前先确认

只有顶部显示 `LLM Agent 模式` 时，助手才真的可用。

规则模式下：

- 输入框会被禁用
- 助手不会调用模型

### 5.2 操作步骤

1. 在右侧输入框输入问题
2. 或点击快捷按钮：
   - 帮我选题
   - 帮我分析视频
   - 帮我写文案
   - 看最近热点
3. 点击发送按钮
4. 等待助手回复

### 5.3 助手会自动理解哪些上下文

助手不仅看你输入的一句话，还会结合页面当前内容：

- 领域
- 方向
- 想法
- 分区
- 风格
- 当前视频链接

所以如果你已经在左侧填好内容，再问“帮我选题”，效果会更完整。

### 5.4 回复里可能包含什么

助手回复除了正文，还可能包含：

- 下一步建议按钮
- 可点击参考视频

这些参考链接可以直接跳转到 B 站页面。

## 6. 语音输入怎么用

当前助手按钮支持两种状态：

- 有文字时：发送
- 没文字时：语音输入

如果你的浏览器支持：

- `SpeechRecognition`
- `webkitSpeechRecognition`

就可以直接点按钮开始语音输入。

如果浏览器不支持，页面会提示你改用文字输入。

## 7. 进度条怎么看

当前页面 3 条主流程都会显示进度条：

- 内容创作
- 视频分析
- 智能助手

页面会同时展示：

- 顶部全局状态
- 当前百分比
- 阶段步骤文案

这是为了让你知道系统正在做什么，不是卡死。

## 8. 清空结果怎么用

顶部右侧有：

- `清空结果`

点击后会清空：

- 内容创作结果
- 视频分析结果
- 视频解析缓存
- 聊天记录

适合在切换测试场景时使用。

## 9. 两种模式下的真实区别

### 9.1 规则模式

特点：

- 内容创作和视频分析仍然可用
- 智能助手不可用
- 不消耗 token

### 9.2 LLM Agent 模式

特点：

- 内容创作走 LLM Agent
- 视频分析走 LLM Agent
- 智能助手可用
- 会消耗 token

## 10. 常见问题

### 10.1 三个输入都没填可以生成吗

不行。

内容创作模块要求：

- 领域 / 方向 / 想法

至少填一项。

### 10.2 为什么粘贴链接后马上出现预览区变化

因为当前逻辑就是：

- 自动解析视频链接
- 先展示预览
- 再点击按钮进入完整分析

### 10.3 为什么聊天按钮有时候是麦克风，有时候是发送

因为当前按钮是复用的：

- 输入框为空 -> 语音输入
- 输入框有文字 -> 发送消息

### 10.4 为什么助手有时会提示回退

如果 LLM Agent 中枢失败，但 fallback 成功，页面会提示：

- 已回退到直接 LLM 生成
- 或已回退到直接 LLM 分析

这说明主流程没有彻底失败，而是换了一条更简单的 LLM 路径。

### 10.5 为什么有 Key 但还是慢

可能原因很多：

- provider 慢
- 模型慢
- 当前复杂分析 prompt 太长
- 超时和重试配置较大

这类问题见：

- [06_Token消耗与AI调用说明.md](D:/agent/docs/06_Token消耗与AI调用说明.md)
- [07_运行模式切换说明.md](D:/agent/docs/07_运行模式切换说明.md)
