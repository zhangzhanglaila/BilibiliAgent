# 前端使用手册

## 1. 打开方式

启动服务：

```bash
python web/app.py
```

浏览器访问：

```text
http://127.0.0.1:8000
```

## 2. 模块一怎么用

适合：

- 还没有确定视频主题
- 想快速生成一版可直接用的文案

操作步骤：

1. 在“你的领域”里输入你的长期方向
2. 在“你想做的方向”里输入当前准备做的内容类型
3. 在“你已有的想法”里输入一个具体想法或问题
4. 选择分区和文案风格
   分区已支持知识、科技、生活、游戏、娱乐，以及美妆颜值、舞蹈、音乐、影视、动漫、美食、Vlog、穿搭、情感、萌宠、运动、汽车、商业职场等更细分类
5. 点击“一键生成选题与文案”

你会得到：

- 系统整理后的创作方向
- 3 个更值得做的选题
- 推荐主选题
- 标题
- 脚本
- 简介
- 标签
- 置顶评论

## 3. 模块二怎么用

适合：

- 视频已经发出去了
- 想看这条视频更像爆款还是低表现
- 想知道下一步该怎么改

操作步骤：

1. 把视频链接粘贴进输入框
2. 点击“一键解析并分析视频”

你会得到：

- 视频基础信息
- 播放、点赞、投币、收藏等数据
- 热门爆款 / 播放偏低 判断
- 对应的分析结果

### 如果系统判断为热门爆款

会重点展示：

- 为什么它能火
- 标题的强点
- 还能继续延展的后续选题

### 如果系统判断为播放偏低

会重点展示：

- 为什么当前数据偏低
- 优化标题建议
- 封面方向建议
- 内容节奏和文案建议
- 可以继续尝试的新选题方向
- 一版新的文案参考

## 4. 常见问题

### 4.1 没有填很多信息，也能生成吗

可以。

模块一里，领域、方向、想法三项至少填一项即可。

### 4.2 视频链接支持什么格式

支持：

- 标准 B 站视频链接
- 常见的 `b23.tv` 短链

### 4.3 如果视频信息解析失败怎么办

通常有三种可能：

1. 当前网络环境无法访问对应视频页面
2. B 站公开接口对该视频返回异常
3. 视频链接本身无效

这时优先检查：

- 链接是否能在浏览器正常打开
- 当前机器是否能访问 B 站

### 4.4 复制按钮有什么用

页面里的标题、脚本、简介、评论等内容都支持直接复制，方便你直接拿去改稿或发布。
