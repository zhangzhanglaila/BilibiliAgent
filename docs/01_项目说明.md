# 项目说明

## 1. 项目定位

这是一个面向 B 站创作者的工作台，核心目标不是做泛化聊天 Demo，而是把创作工作拆成可执行的 4 条链路：

1. 发布前的内容创作
2. 发布后的视频分析与优化
3. 知识库沉淀、同步和检索
4. 在 LLM Agent 模式下的对话式调用

当前 Web 页面已经稳定为：

- 顶部：全局状态 + 运行模式面板
- 左侧：`视频分析 / 内容创作 / 知识库管理` 三模块切换
- 右侧：对话式智能助手

对应入口：

- 前端模板：[web/templates/index.html](../web/templates/index.html)
- 前端脚本：[web/static/app.js](../web/static/app.js)
- Web 服务：[web/app.py](../web/app.py)

## 2. 当前核心能力

### 2.1 视频分析

适用场景：

- 已经发过视频
- 想看这条更像热门爆款还是播放偏低
- 想知道标题、封面、结构、后续题材怎么调

输入：

- B 站视频链接

输出：

- 自动解析后的视频信息 `resolved`
- 表现判断 `performance`
- 规则分析或 LLM 分析结果 `analysis`
- 优化建议 `optimize_result`
- 后续题材建议 `topic_result`
- 参考视频 `reference_videos`
- 对低表现视频额外给出新文案 `copy_result`

### 2.2 内容创作

适用场景：

- 还没有明确选题
- 只有模糊方向、领域、想法
- 想直接拿到标题、脚本、简介、标签和置顶评论

输入：

- 领域 `field`
- 方向 `direction`
- 想法 `idea`
- 分区 `partition`
- 文案风格 `style`

输出：

- 整理后的方向 `normalized_profile`
- 当前核心问题 `seed_topic`
- 推荐选题 `topic_result`
- 推荐主选题 `chosen_topic`
- 完整文案 `copy_result`

### 2.3 知识库管理

适用场景：

- 想导入自己的资料文件
- 想同步当前 B 站热门样本
- 想直接查看当前知识库存储里的内容
- 想按一级分类做知识库检索

当前工作台里的知识库模块主要承担 4 类动作：

- 导入外部知识文件
- 同步 B 站热门样本
- 查看当前库内沉淀内容
- 按一级分类做检索

当前知识库特性：

- 知识库后端优先使用 Chroma，缺依赖时自动回退本地 JSON 存储
- 文本 embedding 优先使用 `sentence-transformers`
- 本地模型不可用时自动回退 deterministic embedding
- 搜索结果会按文档去重，不再把同一篇文档的多个分片分开显示
- 一级分类检索会先召回，再按分类规则过滤
- 发给大模型的是检索命中的 `text / metadata / score`，不是整库文件，也不是向量数组

### 2.4 智能对话助手

适用场景：

- 不想自己判断该点哪个模块
- 想直接用自然语言发任务
- 希望 Agent 自己决定调哪些工具

当前能力：

- 可读取页面上下文
- 可调选题、视频分析、热门快照、知识库检索、Web 搜索、代码执行工具
- 仅在 `LLM Agent 模式` 下可用

## 3. 当前运行模式

项目分成两种运行模式：

### 3.1 无 Key 逻辑模式

特点：

- Web 主界面不消耗 token
- 内容创作、视频分析走规则链路
- 智能助手禁用
- 知识库功能照常可用

### 3.2 LLM Agent 模式

特点：

- 页面运行模式区域可以动态开启
- 内容创作、视频分析、聊天交给 `LLMWorkspaceAgent`
- 助手可自主调用工具
- 会消耗 token

注意：

- 这个开关只影响 Web 主工作台
- CLI 和兼容接口不是完全跟随页面开关

## 4. 当前知识库热门样本同步范围

当前热门样本同步不再只抓内部粗分区，而是抓真实一级分区榜单。

同步来源包括：

- 全站热门榜
- 每周必看
- 入站必刷
- 分区热门榜

当前分区热门榜使用的真实分区包括：

- 番剧
- 国创
- 纪录片
- 动画
- 游戏
- 鬼畜
- 音乐
- 舞蹈
- 知识
- 科技
- 汽车
- 运动
- 动物圈
- 生活
- 娱乐
- 影视
- 电影
- 电视剧
- 时尚

每条入库热门样本当前会保存：

- 榜单来源
- 视频标题
- UP 主
- 分区
- 播放量 / 点赞量
- 评论热词
- 核心优点
- 发布时间点
- BVID
- 榜单链接
- 视频链接

```text
D:\agent
├─ agents/                     业务 Agent
├─ chains/                     路由链和辅助链
├─ docs/                       项目文档
├─ memory/                     长期记忆实现
├─ tools/                      工具层
├─ vector_db/                  知识库数据目录
├─ web/
│  ├─ app.py                   Flask Web 入口
│  ├─ templates/index.html     页面模板
│  └─ static/

├─ graph.py                    LangGraph 编排
├─ knowledge_base.py           知识库后端（Chroma / JSON fallback）
├─ knowledge_sync.py           文件入库和热门样本同步
├─ llm_client.py               LLM 调用封装
├─ main.py                     CLI 入口
└─ requirements.txt            依赖列表
```
